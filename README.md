# VAN_ex — Visual Odometry Exercises

Exercises based on the [KITTI benchmark](http://www.cvlibs.net/datasets/kitti/) visual odometry sequence #00.

## Structure

```
VAN_ex/
├── code/       # Exercise code
├── dataset/    # KITTI data (not tracked in git)
└── docs/       # Documentation and resources
```

## Dataset

Download the KITTI sequence #00 subset and place the zip in `dataset/`, then unzip it there.

The stereo images are located at:
- Left:  `dataset/sequences/00/image_0/`
- Right: `dataset/sequences/00/image_1/`
